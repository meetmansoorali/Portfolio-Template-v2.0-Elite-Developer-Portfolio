# 🚀 Portfolio Template v2.0 — Elite Developer Portfolio

A high-performance, **single-file, one-page portfolio system** engineered for developers, engineers, and creative technologists.

Built with precision — combining **frontend aesthetics**, **backend mindset**, and **conversion-focused UX**.

---

## ✨ Features

* ⚡ Fully responsive modern layout
* 🎯 High-performance UI (no frameworks)
* 🎨 Advanced visual effects (cursor, glitch, particles)
* 📊 Skill visualization system
* 🧠 Structured sections (Hero, About, Expertise, Credentials, Contact)
* 📩 Built-in client intake system (Google Forms integration)
* 🧩 Entire system runs from a single `index.html`
* 🚀 Instant deployment (Netlify / GitHub Pages)

## 🎨 Customization

### 1. Change Personal Info

Edit directly inside `index.html`:

* Name
* Role
* Description
* Social links

---

### 2. Update Skills

Find:

```html
<div class="skill-bar-fill" data-width="96"></div>
```

Modify percentage values.

---

### 3. Replace Images

Search for:

```
i.ibb.co
```

Replace with your own hosted images or local assets.

---

### 4. Client Intake (Contact Form)

Replace the embedded form:

```html
<iframe src="YOUR_GOOGLE_FORM_LINK"></iframe>
```

This acts as your **client intake endpoint**.

---

## 📩 Client Intake System

This template includes a **built-in client intake layer** powered by Google Forms.

* No backend required
* Responses stored in Google Sheets
* Works seamlessly with static hosting

### Data Flow

```
User → Portfolio UI → Embedded Form → Google Forms → Google Sheets
```
